# Run: real 32137 (Palm Coast / Flagler County) leads

This app ships with **no lead data** — leads come from scraping live public
records. Follow these steps to populate it with real ZIP 32137 parcels.

## Prerequisites (one-time)
1. **Network egress** to the data sources (allowlist these hosts in the
   environment's network settings):
   - the Flagler County Property Appraiser ArcGIS host
   - `hazards.fema.gov` (FEMA flood zones)
   - `nominatim.openstreetmap.org`, `overpass-api.de` (water / wetlands)
   - `epqs.nationalmap.gov`, `hydro.nationalmap.gov` (elevation / hydrology)
   - `api.spokeo.com` (skip-trace; optional but needed for phone/email)
2. **Skip-trace key** (for phone/email): `export SPOKEO_API_KEY=...`

## Steps
```bash
# 1. Stage a clean, PII-free copy of the app (optional if you already have one)
tools/stage-eastflorida.sh /tmp/ef
cd /tmp/ef

# 2. VERIFY the Flagler source before the first scrape.
#    The endpoint URL + field names in scraper/counties.js (Flagler) are
#    scaffolded and marked TODO[VERIFY] because egress was blocked when wired.
#    Confirm them against the live service catalog and fix if needed:
curl 'https://gis.flaglercounty.gov/server/rest/services?f=json'   # find the Parcels layer
curl '<.../Parcels/FeatureServer/0?f=json'                         # list field names
#    -> update queryUrl and the a.* field names in COUNTY_SOURCES.Flagler

# 3. Scrape real 32137 parcels into the app (writes leads-data.js)
npm run scrape:32137

# 4. Open it
npm run serve            # http://localhost:8888
```

## What `npm run scrape:32137` does
`node scrape-leads.js --profile scraper/profiles/32137.json --export-app --skiptrace`
- targets **Flagler County**, ZIP **32137**, vacant land only
- applies the client criteria from `scraper/profiles/32137.json`
  (0.25–5 ac, residential/rural-residential, road access, exclude
  wetlands/high-flood; absentee/OOS preferred; equity threshold configurable)
- enriches flood/water/scoring, plus phone/email if `SPOKEO_API_KEY` is set
- writes `leads-data.js` for the map UI

## Notes
- Adjust criteria in `scraper/profiles/32137.json` as they firm up.
- Other ZIPs: copy the profile, set `county`/`zip`, add the county source.
- **Never** commit `leads-data.js` or other scraped contact data to a public
  repo — it contains real owner PII. Keep the repo private.
